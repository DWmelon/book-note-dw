拾记
================================

## About
一个学习性的应用，目的是提供一款`便捷、单肩、高效记录日记`的工具，附加`贴纸库`，以便用户快速找到符合自己日记的贴纸。  

Fork & Star，都无尽感谢。

## Features
* **记录**日常生活
* **备份**日记到云端
* **同步**日记到本地
* **分享**日记到社区
* 提供**贴纸**，点赞/下载

## Framework
* 网络框架：volley
* 图片框架：fresco
* View框架：butterknife
* 数据库框架：greendao
* 异步框架：rxjava
